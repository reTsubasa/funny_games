# funny_games
这是一个给娃做点有意思的玩具的repo

## 目的
DIY一点小玩具


## 原则
- 成本越低越好，能手搓苟且的部件，绝不买
- 手搓难以保证精度或可靠性，而且淘宝买起来很便宜的部件，绝不手搓
- 简单实现优先
- 好的设计总是心有灵犀，电动小车和新能源汽车有啥区别么，多借鉴成熟设计
- 不为了吃顿饺子，而去准备一房间的厨具
- 不卷不卷不卷，玩具要玩的开心，不是自己卷自己

## 已经有的背景知识
- 小学生水准模型知识
- 已经完全还给老师~~也许就没学会过~~的电路知识
- 大学毕设勉强跑通的嵌入式开发知识
- 半桶水~~基本为0的~~开发能力
- 忽高忽低的动手能力
- 超过及格线的互联网冲浪技能
- 自认为还可以的常识
- 人到中年已经大不如前的理解能力
